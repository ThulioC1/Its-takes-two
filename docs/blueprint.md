# **App Name**: Its take two

## Core Features:

- Shared To-Do List: A shared list of activities for the couple to do together, with title, description, status, creation date, and completion date. Includes functionalities to mark as done, edit, and delete items.
- Couple Finances: Record of shared expenses with value, category, payer, date, and optional observation fields. Functionalities include monthly total, filter by category and period, and financial goals visualization.
- Movies and Series List: Shared list of movies and series with name, type, platform, optional link, status (to watch, watching, watched), and date watched. Includes options to edit and remove items.
- Important Dates: Registration of special dates with title, date, type, and optional observation fields. Functionalities include automatic reminders, countdown, and sorting by nearest date.
- Couple's Wall: Private social network-style wall for the couple. Functionalities include creating posts with text and emojis, likes, comments, and date/time of the post.
- Memories Album: Upload of shared photos with image, description, date, and optional location fields. Displayed in a timeline format, with images stored in Firebase Storage.
- Messages / Love Letters: Private messages between the couple. Functionalities include creating loving messages and scheduling a date for the message to become visible, creating surprise messages.
- Couple's Goals: Shared goals with financial, personal, and relationship types. Fields include title, description, progress (%), and status (in progress or completed). Real-time progress update.
- User Authentication: Secure user authentication using Firebase Authentication to manage user accounts and access.
- Data Storage: Utilize Cloud Firestore for real-time data synchronization and structured data storage, organized by couple.

## Style Guidelines:

- Soft color palette to create a romantic feel.
- Clean and modern typography to match the minimalistic style
- Intuitive design, easy to navigate
- Heart-inspired, delicate icons
